# Storybook_example

### Getting started

These instructions will get you a copy of the project up and running on your local machine.

#### Clone

Clone this repo to your local machine using https://github.com/pradnyasd/storybook.git

#### Setup

Now install npm packages

```
$ npm install
```

```
$ npm start
```
